AR Game Project

This artifact is about integrating AR applications into Unity and potentially making AR projects for the purpose of education. I pitched the idea behind this project, it was going to be about a mushroom man saving his world. It had the potential to be integrated into Biology teacher classrooms because the AR markers were going to be educational cards about various types of mushrooms. I learned how to set up an AR webcam system in Unity that is able to track image targets and my modified specialty code enabled and disabled objects using this for various gameplay changes. I also learned how to make mock storyboards for the project. What made this project fun to build for the portions of it I was responsible for was working with the cool Vuforia technology. I felt really accomplished when I got the AR up and running exactly how I wanted it with an extra timer feature that scans the AR markers and then puts them on cooldown.
